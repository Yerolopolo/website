Software alternatives

Operating systems

Avoid Windows and macOS, instead use any GNU/Linux distribution or *BSD variant. If you want a 100% free system check out the FSF endorsed distributions

VPN providers

Don't use one, instead use Tor or I2P, if thats not an option buy IVPN, OVPN or Mullvad VPN preferably via Monero or cash

Web browsers

If you like Firefox switch to LibreWolf, if you like Chrome switch to Ungoogled Chromium, if you want something minimal with Vim keybinds use qutebrowser

Instant messaging services

If you want something modern with multimedia, screenshare, VC and encryption use Matrix with the Element client, if you don't like electron use nheko

Search engines

Avoid closed source search engines, use LibreX, SearXNG or Whoogle, also check out wiby for old-school websites

Git services

If you want to self host use cgit, otherwise use notabug

Code editors (GUI)

Use VSCodium (spyware free VS Code), if you don't like electron use Lite XL

E-mail

Self host, if thats not an option use Daniel Winzen's service, since you can register trough Tor and you can use the web client without javascript
