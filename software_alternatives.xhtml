Software alternatives

VPN providers

Don't use one, instead use TOR or I2P, if thats not an option buy IVPN, OVPN or Mullvad VPN preferably via Monero or cash

Web browsers

If you use Firefox switch to GNU Icecat, if you use Chrome switch to Ungoogled Chromium, if you want something minimal use qutebrowser

Instant messaging services

If you want something modern with multimedia, screenshare and VC use Matrix with the nheko client

Search engines

Avoid closed source search engines, use LibreX, SearXNG or Whoogle, also check out wiby for old-school websites

Git services

If you want to self host use cgit, otherwise use sourcehut
