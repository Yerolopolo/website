Setup

Homeserver

This website along with all of my services are running on my Intel NUC (DC3217IYE) homeserver. I use Trisquel GNU/Linux-libre on it because its stable and it respects my freedom.

The webserver and reverse proxy of my choice is nginx. For renewing TLS/SSL certificates I use certbot since it offers a plugin for nginx which makes things very easy.

You can also access this website and all of my services on my Tor mirror.


Laptop

I take a librebooted Thinkpad T60 to school for coding. I'm currently using Parabola GNU/Linux-libre on it, therefore my computer is 100% free. The issue with almost all GNU/Linux distributions is the Linux kernel since it ships with non free firmware.


Desktop

My main PC is a self-built desktop, it has an RX 6800 GPU, Ryzen 5 5600X CPU and 16GB of RAM. Due to the decent specs of the PC I use Gentoo GNU/Linux on it.


Software

My window manager of choice is dwm with st and dmenu. For my browsers I use GNU Icecat and Ungoogled Chromium, because these don't track me like Firefox or Chromium. I use lite as my code editor and I use Cinny as my Matrix client.
